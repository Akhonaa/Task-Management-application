# Task Management Application

This Task Management Application allows users to log in, add tasks, and generate various reports on tasks. It provides functionalities such as searching for tasks, deleting tasks, and displaying specific reports based on task status and duration.

## Features

- **User Authentication**: Users must log in with a valid username and password.
- **Add Tasks**: Users can add multiple tasks, each with details like task name, description, developer details, duration, and status.
- **View Reports**: Users can view reports of all tasks, filter tasks based on status, find the longest task, search tasks by name or developer, and delete tasks by name.
- **Test Data**: The application includes predefined test data for demonstration and testing purposes.




## Usage

1. **Run the Application**: Execute the `App.java` file.
2. **Log In**: Enter a valid username and password.
3. **Main Menu**: After logging in, you can:
   - Add tasks
   - Show reports
   - Display tasks with specific statuses
   - Search tasks
   - Delete tasks
   - Quit the application
